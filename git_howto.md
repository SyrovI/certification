## Подсказки для языка MarkDown
*Создать репозиторий*
```sh
git init
```
*Добавить изменения*
```sh
git add
```
**Создать коммит**
```sh
git commit -m<message>
```
**Показать коммиты**
```sh
git log
```
**Показать сокращенные коммиты**
```sh
git log --online
```
**Перейти на коммит**
```sh
git checkout <№ commit>
```
