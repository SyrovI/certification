## Подсказки для языка MarkDown
*Создать репозиторий*
```sh
git init
```
*Добавить изменения*
```sh
git add
```
**Создать коммит**
```sh
git commit -m<message>
```
**Показать коммиты**
```sh
git log
```
**Показать сокращенные коммиты**
```sh
git log --online
```
**Перейти на коммит**
```sh
git checkout <№ commit>
```
**Вернуться к актуальному состоянию репозитория**
```sh
git checkout master
```
**Показать отличия коммитов**
```sh
git diff <№ commit>
```
### Команда смены директории
```sh
cd c:\folder_name
```
### Удаление файла в Windows:
```sh
del <filename>
```

```

 ```
